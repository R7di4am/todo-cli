# Todo List App

A simple Todo List application built using Python. 

## Features:
- List all todos
- Add a new todo
- Edit an existing todo
- Delete a todo

## Usage:
1. Use command `1` to list all todos.
2. Use command `2` to add a new todo.
3. Use command `3` to delete a todo.
4. Use command `4` to edit a todo.
